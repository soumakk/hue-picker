# 🎨 Hue Picker

A minimal, developer-friendly color picker

## 🖼️ Preview

![screenshot](/public/screenshot.png)

## ✨ Features

- 🎨 Fine-tune colors using RGB & HSV sliders
- 🔄 Seamlessly convert between formats
- 🌈 Instantly generate tints and shades
- ♿ Built-in contrast checker for accessible color combinations
- 🧾 Copy colors in multiple formats (raw values, CSS, JSON)
- ⚡ Clean, fast, and responsive interface

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Base UI
- [react-color-palette](https://github.com/Wondermarin/react-color-palette)

## 🚀 Getting Started

```bash
git clone https://github.com/soumakk/hue-picker.git
cd hue-picker
npm install
npm run dev
```
